# 栈

1. 栈也是一个受限的线性结构
   栈只允许从`一端`入栈、`一端`出栈，先入栈的元素会最后出栈，最后入栈的元素最先出栈

2. 栈的特点: 先进后出

3. 栈常见的方法

```TypeScript
/* 入栈: 尾部添加元素 */
push(element: T): void { }
/* 出栈: 头部删除元素，并返回删除的元素 */
pop(): T | undefined {}
/* 查看栈尾的元素，并返回查看的元素 */
peek(): T | undefined {}
/* 判断栈中是否为空 */
isEmpty(): boolean {}
/* 获取当前栈中的个数 */
size(): number {}
```
